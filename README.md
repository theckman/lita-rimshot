INSTALLATION
------------
Add this line to your Lita bot's Gemfile:

```Ruby
gem 'lita-rimshot'
```

COMMANDS
--------
* `rimshot` - gives you an instant rimshot for a joke

USAGE
-----
```
>> Lita rimshot
Here ya go! http://instantrimshot.com/classic/?sound=rimshot
```
